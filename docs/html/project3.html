Classiciation metrics

class classification_metrics.Metrics(y_true, y_pred, classes)[source]

Class for calculating the accuracy, precision, and recall between predicted and true labels.

\[\begin{split}\begin{align} \mathrm{Accuracy}&=\mathrm{\frac{Correct\,predictions}{All\,predictions}}\\ \mathrm{Precision}&=\mathrm{\frac{TP}{TP+FP}}\\ \mathrm{Recall}&=\mathrm{\frac{TP}{TP+FN}} \end{align}\end{split}\]

where \(TP,FP,FN\) are true positive, false positive, and false negative, respectively.

Parameters:

  • y_true (array_like) – True labels.

  • y_pred (array_like) – Predicted labels.

  • classes (array_like) – Class or classes to evaluate the metrics against.

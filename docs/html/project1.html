Regression class#

class src.Regression(x_data, y_data, z_data=None)#

Regression class for Ordinary Least Square (OLS), Ridge (ridge), and Lasso (lasso) regression.

x_data, y_data

Data along the x and y axes.

Type:

array_like

z_data#

If the function describing the data is 2D, the z_data parameter describe the data along the z axis.

Type:

array_like, default: None

Regression methods#

Regression.OLS(n_poly, identity_test=False)#

Ordinary least square regression method, ie.:

\[\begin{split}\begin{align} \tilde\beta &= (X^TX)^{-1}X^Ty \\ \tilde y &= X\tilde\beta \end{align}\end{split}\]

with a user-defined polynomial degree.

Parameters:

  • n_poly (int) – Polynomial degree.

  • identity_test (bool, default: False) – If True, the method performs a test to see if the implementation is correct, ie. if the design matrix is the identity matrix, the mean square error should be 0.

Regression.ridge(lambda_min, lambda_max, poly_deg, n_lambda)#

Perform Ridge regression on the provided data.

\[\begin{split}\begin{align} \hat\beta&=(X^TX+\lambda\mathbb{I})^{-1}X^T y \\ \tilde y&=X\hat\beta \end{align}\end{split}\]
Parameters:

  • lambda_min (float) – Lowest/highest value for \(\log_{10}\lambda\).

  • lambda_max (float) – Lowest/highest value for \(\log_{10}\lambda\).

  • poly_deg (int) – Order of the polynomial to fit.

  • n_lambda (int) – Number of \(\lambda\)-elements to compute.

Regression.plot_evolution(model, figname, add_lasso=True)#

Plot the evolution of the Mean Squared Error (MSE), R2 score, and the values for the optimal parameters \(\hat\beta\).

Parameters:

  • model ({'OLS', 'ridge', 'lasso'}) –

    Which model to use.

    • ’OLS’: The curves will be as functions of the polynomial degree.

    • ’ridge’ or ‘lasso’: The curves will be as functions of \(\lambda\).

  • figname (str) – For saving the figure.

  • add_lasso (bool, default: True) – Option to add the lasso model to the figure for ridge. If so, the figure will show only the MSE and R2 score for both models.

src.set_size(width='col', scale=1.0, subplot=(1, 1))#

Set the size of a figure where the height/wifdth equals the golden ratio.

Parameters:

  • width ({'col', 'text'} or float, optional) –

    Width of the figure in pt. Possible values:

    • ’col’ (default): 255.46837

    • ’text’ : 528.93675

  • scale (float, default: 1.0) – How to scale the height of the figure, ie. figsize=(width, height * scale)

  • subplot (tuple, default: (1, 1)) – How to scale the figure size based on the number of subplots: figsize=(width, height * subplot[0] / subplot[1])

Returns:

Dimension of the figure, ie. (width, height * scale * subplot[0] / subplot[1])

Return type:

tuple

Notes

The built-in values for width and height are the column width and text width in REVTeX document class. To obtain the appropriate values for your document, run the commands \the\columnwidth and \the\textwidth in your dobument body.

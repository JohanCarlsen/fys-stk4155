class src.Regression(x_data, y_data, z_data=None)

Regression class for Ordinary Least Square (OLS), Ridge (ridge), and Lasse (lasso) regression.

Parameters:

x_data, y_dataarray_like

Data along the x and y axes.

z_dataarray_like, default=`None`

If the function describing the data is 2D, the z_data parameter describe the data along the z axis.

OLS(n_poly, identity_test=False)

Ordinary least square regression method, ie.:

\[\begin{split}\tilde\beta = (X^TX)^{-1}X^Ty \\ \tilde y = X\tilde\beta\end{split}\]

with a user-defined polynomial degree.

Parameters:

n_polyint

Polynomial degree.

identity_testbool, default=`False`

If True, the method performs a test to see if the implementation is correct, ie. if the design matrix is the identity matrix, the mean square error should be 0.

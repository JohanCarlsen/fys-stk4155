class ffnn.NeuralNetwork(input_size, hidden_sizes, output_size, eta, alpha, hidden_activation, output_activation, cost_function, epochs, batch_size, solver, random_weights=False)[source]ÔÉÅ

Artificial neural network class for regression and classification.

Parameters:

  • input_size (int) ‚Äì Nodes in the input layer.

  • hidden_sizes (list of ints) ‚Äì Structure of the hidden layers. E.g. [10, 20, 10] will create 3 hidden layers consisting of 10, 20, and 10 hidden neurons, respectively.

  • output_size (int) ‚Äì Nodes in the output layer.

  • eta (float) ‚Äì Learning rate.

  • alpha (float) ‚Äì L2 regularization parameter.

  • hidden_activation ({'sigmoid, relu, lrelu'}) ‚Äì

    Activation function for the hidden layers. Note that the weights are initialized depending on the activation function as:

    • ‚Äôsigmoid‚Äô : XavierInitializer.

    • ‚Äôrelu‚Äô : HeInitializer.

    • ‚Äôlrelu‚Äô : LeCunInitializer.

  • output_activation ({'sigmoid', 'relu', 'lrelu', 'linear'}) ‚Äì Activation function for the output layer. If other than linear, classification is assumed.

  • cost_function ({'mse', 'cross', 'log'}) ‚Äì

    Cost or loss function.

    • ‚Äômse‚Äô : Mean squared error.

    • ‚Äôcross‚Äô : Cross entropy.

    • ‚Äôlog‚Äô : Log-loss.

  • epochs (int) ‚Äì Number of iterations to perform.

  • batch_size (int) ‚Äì Minibatch size to use for stochastic gradient descent. If batch_size == n_datapoints, regular gradient descent is used.

  • solver ({'adam', 'constant'}, str) ‚Äì Which optimizer to use. ADAM is the best.

  • random_weights (boolean, optional) ‚Äì Weather to overwrite the weight initialization. If True, the weights will be initialized from the normal distribution. If False (default), the activation of the hidden layers controls this.

Logistic regression class

class logreg.LogisticRegression(eta, alpha, n_epochs, tol=0.0001)[source]ÔÉÅ

Logistic regression class for binary classification.

The sigmoid function is used on the predicted outcome:

\[\begin{split}\begin{align} \tilde y &= \mathbf X^T \mathbf\beta\\ \sigma(\tilde y) &= \frac{1}{1+e^{-\tilde y}} \end{align}\end{split}\]

The cost function is given by the log-loss:

\[\begin{align} C(\beta)=-\frac{1}{n}\sum_n\left[y_n\log(\tilde y_n+\delta)+(1-\tilde y_n)\log(1-\tilde y_n+\delta)\right] \end{align}\]
Parameters:

  • eta (float) ‚Äì Learning rate (step size).

  • alpha (float) ‚Äì L2 regularization parameter.

  • n_epochs (int) ‚Äì Number of epochs (iterations) to perform.

  • tol (float, optional) ‚Äì Tolerance of how equal the gradients are between runs (default: 10^-4).
